Welcome to the Munro API

This is a work in progress, with more data & routes to be added.

The current routes available are:

  • /munros - View all Munros
  • /munros/name/:name - View Munros by Name
  • /munros/smcid/:smcid - View Munros by SMC ID
  • /munros/region/:region - View Munros by Region

No API key is required.
